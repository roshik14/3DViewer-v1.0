s21_3DViewer_v1.0

by caquaman, sperses and mtickler

Information

Implementation of project 3DViewer_v1.0 using C language and GTK+ library. This is a program to view 3D wireframe models. The models themselves load from .obj files and viewable on the screen with the ability to rotate, move and scale. The program allows you to set custom settings regarding the display of the 3D model. In addition, the recording of the received image into files of bmp and jpeg formats is implemented. The program also allows you to record small screencasts in a .gif file

Usage

Run program without installation

To run program without install just type 

make run

Rotate and move object

To rotate or move object just type x,y,z values and click the button Rotate or Move

Scale object

To scale object just type scale value and click the button Scale. The program does not accept negative values. To constriction object , for example by 5, just type 0.5

Record object

To record small screencast in a .gif file click the button Record, then click the button Save gif Then move, rotate, scale, or do what you want to record screencast. To finish recording click the button Record again and then click the button Stop

Installation

Install:

To install the program type 

make install
Then go to folder where application was installed
(Usually this is /Users/Username/Desktop/3DViewer/)

Then type 
./viewer

Uninstallation

Uninstall

To uninstall program type 

make uninstall
